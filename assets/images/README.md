# Carpeta de Imágenes

Sube aquí las siguientes imágenes para la landing page:

## Imágenes requeridas:

1. **logo.png** - Logo de VecindApp
2. **qr-download.png** - Código QR para descargar la APK de Android
3. **app-screenshot.png** - Captura de pantalla de la aplicación en uso
4. **duoc-logo.png** (opcional) - Logo de Duoc UC

## Notas:

- El código QR debe apuntar a la URL de descarga de la APK
- Las imágenes se mostrarán automáticamente cuando se suban a esta carpeta
- Si faltan imágenes, la página mostrará placeholders
